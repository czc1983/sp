## 下一步主线

目标不是普通白膜，而是“去身份化光影白模控制视频”：

```text
原视频
-> 分镜切开
-> 生成深度/白模
-> 去掉原人物身份信息
-> 保留镜头、构图、动作、光影、遮挡
-> 输出 Seedance 可用控制视频
```

## 白膜资产要拆成几种输出

1. `BG_ON`：原背景 + 光影 + 去身份化白模人物。
   - 用于现在的人物替换测试。
   - Seedance 主要看这个，不要直接看原视频人物外观。

2. `BG_OFF`：透明/黑底/中性背景 + 去身份化白模人物。
   - 用于将来换背景。
   - 背景不锁死，人物动作和遮挡仍保留。

3. `clean_plate`：清场背景。
   - 把原人物擦掉。
   - 用于未来背景修复、换背景、合成预览。

4. `internal_id_mask`：内部人物 ID 跟踪。
   - 可以是彩色，但不要直接喂 Seedance。
   - 只给程序判断谁是谁，避免颜色污染生成。
