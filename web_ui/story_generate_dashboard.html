2. 逐镜头分解

得到镜头级分镜：每个镜头的起止时间、人物、动作、场景、情绪和生成提示词。

3. 按分镜切片

切片边界跟随分镜和镜头，同一镜头内保留连续动作。

4. 重新生成

接入视频生成模型生成新片段，再按剧情顺序剪回完整片子。
